# Project Context
Este es un proyecto de plataforma SaaS para la gestión de invitaciones digitales y planificación de eventos. 
Utilizamos Next.js (App Router), Firebase (Firestore/Auth), Zustand para el estado, Tailwind CSS y Framer Motion.
El código está dividido estrictamente en módulos de "admin" (Backoffice) y "front" (Invitados finales). 
Siempre priorizamos el rendimiento en móviles y la limpieza arquitectónica (Feature-Sliced Design).
