# Skill: Evaluate Front Code
Description: Audita las vistas públicas y componentes de la invitación digital asegurando un enfoque Mobile-First, alta conversión y rendimiento óptimo.

## Instructions
1. Lee silenciosamente los siguientes blueprints:
   - `.docs/rules/clean-architecture.md`
   - `.docs/rules/front-ui-rules.md`
2. Analiza el código del archivo o directorio proporcionado.
3. Verifica la separación de responsabilidades (Clean Architecture).
4. Evalúa la UI y el Rendimiento:
   - ¿Es Mobile-First?
   - ¿Aprovecha Server Components de Next.js donde es posible para mejorar LCP?
   - ¿Usa Framer Motion de forma fluida y sin bloquear el hilo principal?
5. Devuelve un análisis crítico enfocado en la experiencia del invitado final.
6. Escribe la versión optimizada del código.

## Constraints
- Asume que el usuario final NO está autenticado (depende de tokens en la URL).
- No introduzcas tablas de datos complejas ni elementos propios del panel de administración.
